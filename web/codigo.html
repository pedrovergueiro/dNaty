🔄 Fluxo Completo de Execução

Este é o caminho que o código percorre quando você roda um experimento. Cada seta mostra quem chama quem.
1
exp1_mnist.py → main()
Ponto de entrada. Define configurações (seeds, gerações, população). Chama run_dnaty_seed() para cada seed.
2
run_dnaty_seed() → FastDataset()
Carrega o dataset MNIST inteiro na RAM. 60K imagens viram tensores PyTorch. Zero I/O depois disso.
3
run_dnaty_seed() → DnatyEvolver()
Cria o evolver com os hiperparâmetros. Inicializa a população de 12 redes e a memória episódica compartilhada.
4
DnatyEvolver.run() → loop de gerações
Para cada geração: mutação → treino → avaliação → seleção → atualiza memória. Repete até early stop ou G gerações.
5
_mutate_population() → apply_operator()
Para cada indivíduo, consulta a memória para escolher o operador mais provável. Aplica o operador na rede.
6
_train_parallel() → local_train()
Treina cada indivíduo mutado por T_local epochs. Retorna (loss_antes, loss_depois, grad_norm).
7
_eval_population() → evaluate()
Avalia todos os indivíduos no dataset de validação. Usa torch.inference_mode() para máxima velocidade.
8
nsga2_select() → fronts de Pareto
Seleciona os N melhores indivíduos balanceando acurácia, número de parâmetros e sharpness.
9
_update_memory() → EpisodicMemory.update()
Para cada indivíduo que melhorou em relação ao pai, registra a experiência na memória.
10
Salva resultados → results/exp1_results.json
JSON com acurácia, params, histórico de convergência, validação do Teorema 1.

memory.py — EpisodicMemory

O "caderno de anotações" do dNaty. Guarda quais operadores funcionaram e com que intensidade.

class EpisodicMemory:
    def __init__(self, max_size=500, decay_gamma=0.99):
        self._scores = {}  # scores por operador — O(1) por update

    def update(self, exp: Experience):
        # Decai todos os scores por gamma (memória esquece gradualmente)
        for op in self._scores:
            self._scores[op] *= self.gamma
        # Adiciona o impacto da nova experiência
        if exp.impact > 0:
            self._scores[exp.operator] += exp.impact

    def query_mutation_probs(self, operators):
        # Retorna distribuição softmax sobre os operadores
        # Operadores com mais impacto acumulado têm maior probabilidade
        vals = [self._scores.get(op, 0.0) for op in operators]
        return softmax(vals)

class Experience:
    @property
    def impact(self):
        # Só conta se a mutação MELHOROU o modelo (não-circular)
        if self.delta_loss >= 0: return 0.0
        return abs(self.delta_loss) * self.gradient_norm

individual.py — Individual

Um "indivíduo" na população. Contém o modelo (rede neural) + sua memória episódica própria.

class Individual:
    def __init__(self, model, memory=None):
        self.model   = model    # DynamicMLP ou DynamicCNN
        self.memory  = memory   # EpisodicMemory própria
        self.acc     = 0.0      # acurácia atual
        self.fitness = (0,0,0)  # (acc, -params, -sharpness)
        self.last_op = 'init'   # último operador aplicado

    def clone(self):
        # Cria cópia profunda — necessário antes de mutar
        return Individual(deepcopy(self.model), deepcopy(self.memory))

O coração do dNaty. Implementa o Algorithm 1 completo com 6 fases por geração.

class DnatyEvolver:
    def run(self, train_loader, val_loader,
            early_stop_patience=8, early_stop_min_delta=1e-4):

        self._init_population()  # cria N indivíduos iniciais
        fitnesses = self._eval_population(population, val_loader)

        for gen in range(1, n_generations+1):

            # FASE 1: Mutação guiada pela memória
            prev_accs = [ind.acc for ind in population]  # acurácia dos pais
            mutated = self._mutate_population(population)
            # → consulta memória → escolhe operador → aplica → retorna novo ind

            # FASE 2: Treino local (Lema 2)
            for ind in mutated:
                lb, la, gn = local_train(ind, train_loader, ...)
                # lb = loss antes, la = loss depois, gn = grad_norm
            delta_grad = mean(lb - la)  # sempre positivo

            # FASE 3: Avaliação multiobjetivo
            mut_fitnesses = self._eval_population(mutated, val_loader)

            # FASE 4: Seleção NSGA-II
            combined = population + mutated
            population = nsga2_select(combined, fitnesses, n_pop)

            # FASE 5: Atualiza memória (Lema 1)

            # → se melhorou: registra Experience na memória compartilhada

            # Early stopping

def fast_non_dominated_sort(fitnesses):
    # Vetorizado com numpy broadcasting — O(n²) mas sem loops Python
    F = np.array(fitnesses)
    ge = F[:, None, :] >= F[None, :, :]  # i >= j em cada objetivo
    gt = F[:, None, :] >  F[None, :, :]  # i >  j em algum objetivo
    dominates = ge.all(axis=2) & gt.any(axis=2)

def local_train(ind, loader, n_epochs=4, lr=2e-3, batch_size=512):
    model = ind.model.to(device)

fast_dataset.py — FastDataset

A otimização mais importante da v5. Carrega o dataset inteiro na RAM uma única vez.

class FastDataset:
    def __init__(self, name='MNIST', device='cpu', train_subset=None):
        # Carrega TUDO na memória via DataLoader com batch=len(dataset)
        loader = DataLoader(dataset, batch_size=len(dataset))
        x, y = next(iter(loader))  # uma única leitura de disco
        self.train_x = x.flatten(1).to(device)  # (60000, 784) na RAM
        self.train_y = y.to(device)

    def get_train_batch(self, batch_size=512):
        # Indexação direta em tensor — O(1), zero I/O
        idx = torch.randint(0, self.n_train, (batch_size,))
        return self.train_x[idx], self.train_y[idx]

    def get_val(self):
        return self.val_x, self.val_y  # tensor completo, zero cópia

Resultado: DataLoader padrão = 2.3s/epoch. FastDataset = 0.08s/epoch. 28× mais rápido.
